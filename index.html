About Me

I am a cybersecurity researcher. My general research interests are in Security and Privacy of Smart Devices. They encompass following specific areas:

  • Usable security for smart hand-held devices
  • In-vehicle security and intrusion detection for connected/autonomous cars
  • Self-aware operating systems for mobiles and smart cars
  • Software engineering for ensuring application and process security
  • Security of the communication between smart/IoT devices and cloud/fog networks
  • Using machine learning techniques for security analytics and automation

Selected Publications

Droid Mood Swing (DMS): Automatic Security Modes based on Contexts

Droid Mood Swing (DMS) is an operating system component that applies different security policies to detected security modes automatically. DMS uses a context manager that tracks the context of the phone from the available sensors. DMS then determines the security mode from the contexts and can impose a number of security measures.
S. Iqbal*, M. Zulkernine
In ISC, 2017.
